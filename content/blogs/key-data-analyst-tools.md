As you learn, data analysts commonly use spreadsheets, query languages, and visualization tools in their work. In this reading, you'll get to know more about each of them, including when and why they are essential in data analytics.

### Spreadsheets
Spreadsheets are essential tools for data analysts to gather and arrange data. You'll likely use Microsoft Excel and Google Sheets a lot in your future role as a data analyst.

Spreadsheets help you:
1. Collect, store, organize, and sort information.
2. Find patterns and combine data for each data project.
3. Create impressive data visualizations, such as graphs and charts.

### Databases and Query Languages
A database is a collection of organized data stored in a computer system. Some popular programs that use Structured Query Language (SQL) are MySQL, Microsoft SQL Server, and BigQuery.

Query languages:
1. Help analysts find specific information from a database(s).
2. Make it easier to learn and understand the requests made to databases.
3. Enable analysts to select, create, add, or download data from a database for analysis.

### Visualization Tools
Data analysts use various visualization tools, such as graphs, maps, tables, and charts. Two popular ones are Tableau and Looker.

These tools:
1. Turn complex data into a story that people can easily understand.
2. Help stakeholders reach conclusions for informed decisions and effective business strategies.
3. Come with multiple features:
- Tableau allows users to create interactive graphs in dashboards and worksheets with simple drag-and-drop.
- Looker connects directly to a database, making it easy to visualize your data with the tool you prefer.

In a data analyst career, you'll also work with programming languages like R and Python, widely used for statistical analysis, visualization, and other data tasks.
